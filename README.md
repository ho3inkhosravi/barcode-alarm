# ⏰ آلارم بارکد (Barcode Alarm)

اپلیکیشن آلارم هوشمند که برای قطع شدن، باید بارکد محصول را اسکن کنید!

## ✨ امکانات

- 📷 **اسکن بارکد** — برای قطع آلارم باید بارکد محصول واقعی را اسکن کنید
- ⏱️ **تایمر ۳۰ ثانیه** — فرصت کافی برای رسیدن به بارکد
- ⏰ **انتخاب ساعت** — با TimePicker آسان
- 🎵 **صدا و لرزش** — آلارم با صدای بلند و لرزش
- 🔄 **تکرار روزانه** — تنظیم برای روزهای هفته
- 😴 **چرت (Snooze)** — تعویق آلارم
- 🌙 **رابط فارسی راست‌چین** — طراحی زیبا

## 📥 دانلود APK

به تب **Actions** در GitHub بروید، آخرین build را انتخاب کنید و فایل APK را از بخش Artifacts دانلود کنید.
